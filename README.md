疫情爆发后，很多公司为了有效减少人员聚集，阻断疫情传播，允许员工在家远程办公。部分互联网公司具备远程办公的基础能力，但是还有很多公司并不具有相应的能力，尤其很多传统行业的公司可能不具备相应的能力。

那么在家里如何远程办公呢，今天就来分享一下远程办公的各类软件、工具和管理方式以及建议，以提高企业的远程管理、沟通和协作效率。管理包括到团队管理和项目管理；沟通主要介绍各种沟通工具以及优劣对比；在线协作主要包括日程&日历管理、在线会议（包括视频会议）、在线文档共享编辑、传输存储、屏幕录制&共享。好消息是目前大部分产品都有较长时间的免费试用期限或者试用期限。

期望在这个特殊的时期，帮忙到更多的公司和朋友能够尽快投入到社会生产中来。各位在企业远程办公中遇到什么问题的话，都可以在文章底部提供的渠道寻求帮助。

## 远程办公的挑战

**远程办公最大的挑战就是，远程之下，管理、沟通、协作效率会大大下降**。下面就来拆解，如何通过理论和工具解决这些问题，提高远程管理、沟通和协作效率。

## 管理

### 团队人员管理

远程工作，首先团队管理能力下降，因为见不到员工，不能快速安排工作、调整节奏，也不清楚员工在做什么，不知道员工遇到了什么问题，还不能当面指导员工工作。

远程工作不等于放羊，这里有几个建议：

- 设立明确的团队目标，因为团队有着一致的目标，减少不必要的冲突和误解，需要领导者花费时间统一目标和使命。
- 加强定期沟通，沟通包含好几个方面，后面会重点介绍。
- 团队早会，沟通一下今天要做什么事情
- 每日下班邮件汇报，汇报当天的进度

### 项目管理

对于企业而言，比较担心员工的工作进度不可控，项目的进度失控，以及团队的协同失控，这是很多企业管理者，关于项目开发，项目运营最大的噩梦。对于远程工程工作的管理者来说更是至关重要。

因此需要制定全面的项目管理计划，为每个中大型建立一个全局的项目「地图」，制定明确的项目计划和执行跟踪。力求做到所有同学，看到这个地图后，就能够清楚的知道现在是什么情况，需要的资源的链接在哪，负责人是谁，风险点在哪。首先推荐几个在线项目管理工具。

## 沟通

在远程工作最大的挑战就是沟通了。沟通的最大的意义在于：信息对称。大白话就是大家需要相互知道自己该干嘛，团队该干嘛，正在干嘛。在公司，一般通过会议、总结实现。远程主要通过聊天工具、远程会议、视频会议实现。

远程办公我们首先要解决的是我们要建立远程可操作的更加清晰、高效的目标和计划管理。这里有几个建议：

- **加强定期沟通**：在团队管理、项目管理中，一定要做好定期沟通，尽早达成一致，暴露问题。沟通的过程中需要确保信息的传递和反馈的即时性，有效性和完整性。
- **沟通之前准备好上下文环境**：使用简单的几句话描述一下背景、问题以及可行的方案，尽量做到不了解背景的同学在阅读了以后也能够了解到当前的问题。
- **使用一段文字描述问题**：而不是微信语音消息（这个太浪费时间了），也不要连续发送好几条消息，发送之前整理好一段话，也就是上面说的上下文环境，然后在自己阅读一次，确保通畅。
- **重要的结论和方案文档化**：面对面交谈、电话语音、微信 虽然是比较实时的反馈工具，但是只有文档是可以把重要信息给结构化的，而且写文档其实是比起前面的方式来说是更为深度的思考，因为会让你自己审视自己的想法。所以，对于一些重要结论和方案，最好都以文档化的方式进行。扩展一下其实 **所有的产出都应该文档化**。

#### Slack

Slack 有一个很强大的功能，可以作为各种工具的 Message Hub，尽可能做到在一个地方就能一目了然事情的状态。

同时也有个比较严重的问题，有些同学看着微信，有些同学看着 Slack，这就导致了信息不对称，这种情况再有外部客户的时候更加严重，但是内部沟通的话，Slack 是我心中最好的 IM 工具。

## 在线协作

工欲善其事，必先利其器，使用良好的工具会让事情事半功倍。这里主要介绍各种在线协作的工具和软件。

### 日历和日程管理

![calendar2](https://tva1.sinaimg.cn/large/006tNbRwgy1gbh1c8355nj327y0s4wl5.jpg)

**善用 Calendar。**我建议研发团队内部将 Calendar 可以别人可见（这点上财务，商务，高管团队需要酌情考虑），通过订阅和你相关的同事的 Calendar 是一个也是一个很好的信息拉平渠道。可以了解到团队和协作同事的日程安排，有利于安排远程会议。

[头条飞书](https://mp.weixin.qq.com/s/6_Z6Bus9YGiDNMcHtZdQKg)、Google calendar、Office 365 Calendar 都能提供这个服务。

### 在线会议

#### 腾讯会议

是一款互联网在线会议工具，疫情期间免费开放300人实时音频和视频交流能力。

支持电脑/手机客户端、小程序、电话**四大方式一键入会**；会议期间，提供控制禁言、在线文档协作、实时屏幕共享等**多个会议管控功能**；还有**美颜和背景虚化**等功能，帮你避免在家办公的尴尬局面。电脑客户端请https://meeting.qq.com下载。

#### 关于会议的几个建议

1. **尽量少开大会长会**。Amazon CEO 贝索斯 的「两个 Pizza 原则」：团队的大小只能到两张披萨就能喂饱的大小，也同样适应于开会。
2. **会议前发 Calendar**，会议后形成 **会议纪要** 发给参会的人，并记录在 Wiki 中，因为有很多人并不会参会，但是有必要让他们知晓会议内容。

## 相关事项

- 全员动员：远程协作是一种生产模式，区别于集中生产，需要个人主动线上沟通交流，而不是等别人邮件提醒，组织会议，所以一定需要全员动员，让团队成员理解、认同这种协作方式。
- Owner 文化：Ownership 能激发出个人的自豪感和使命感，使之主动自觉地与公司齐心协力朝着既定的目标前进；能激发出员工的凝聚力和创造力，使之真正把公司所当成自己的平台，在本职岗位上激情本分，用自己的专业能力和智慧为公司发展献力。
- 有任何问题可以提出评论和 issue，特别是非互联网行业的人员，我们和乐意帮助大家解决远程协作方面遇到的问题。
