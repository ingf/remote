疫情爆发后，很多公司都先不用上班，在家远程办公。其实部分互联网公司具备远程办公的基础能力，但是还有很多公司并不具有相应的能力，由其很多传统行业的公司可能有比较大的难度。那么在家里如何远程办公呢，今天就来分享一下，如何在家里，高效远程办公。



**远程办公最大的挑战就是，远程之下，管理、沟通、协作效率会大大下降**。下面就来拆解，如何通过理论和工具解决这些问题。
